# POS 工作流程（目前進度版）

## 已完成（2026-02-27）
1. 環境建置：Ubuntu 22.04 + MySQL + Nginx + PHP-FPM
2. Android Studio 專案建立與實機部署（Samsung hlte）
3. ERP導向資料表建立（23 tables）
4. POS API Phase 1：
   - 商品查詢 / 掃碼查詢
   - 建立訂單（收款）
   - 扣庫存與庫存異動紀錄
   - 盤點 API（實盤回寫）
   - 報表 API（日營收、支付占比）
   - 分店同步 API（冪等鍵）
5. Android App Phase 1：
   - 掃碼銷售 + 送單
   - 掃碼盤點 + 實盤數輸入 + 確認送出
   - 今日/昨天/近7天/近15天/近30天 報表查詢

## 下一步（下一次開工）
1. 新增店別篩選報表
2. 報表日期自選（起訖日期）
3. 盤點明細可編輯指定品項（不只刪最後）
4. POS 銷售明細頁（可調整數量/刪品項）
5. 多店同步格式（批次訂單上傳 schema v1）
